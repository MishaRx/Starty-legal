Основные функции расширенияCore Features of the Extension

Эти функции работают как часть базовой работы расширения — согласие на них даётся самим фактом установки и использования расширения по его прямому назначению. В текущей версии все они обрабатывают данные исключительно локально, ничего никуда не передавая:

These features are part of the extension's basic operation — consent to them is given by the fact of installing and using the extension as intended. In the current version, all of them process data exclusively locally, without transmitting anything anywhere:

Отображение значков сайтов. При сохранении ссылки её домен используется для получения иконки (favicon) сайта из локальной базы браузера (разрешение «favicon»). Обрабатывается локально, никуда не передаётся.Displaying site icons. When you save a link, its domain is used to fetch the site's icon (favicon) from the browser's local database (the "favicon" permission). Processed locally, never transmitted anywhere.

Импорт закладок. Если при первой настройке вы выбираете «Импортировать закладки» — расширение считывает заголовки и адреса ваших закладок, чтобы предложить выбрать, какие добавить как плитки. Обрабатывается локально, никуда не передаётся.Bookmark import. If you choose "Import bookmarks" during initial setup, the extension reads the titles and addresses of your bookmarks to let you pick which ones to add as tiles. Processed locally, never transmitted anywhere.

Подсказки недавних сайтов. При открытии формы добавления новой ссылки расширение может показать недавно закрытые вкладки как быстрые варианты. Обрабатывается локально, никуда не передаётся.Recent-site suggestions. When opening the add-link form, the extension may show recently closed tabs as quick options. Processed locally, never transmitted anywhere.

Подсказки при поиске. Показываются из встроенного в расширение словаря частых запросов — сравнение с тем, что вы печатаете, происходит на устройстве. Никуда не передаётся, внешние сервисы не используются.Search suggestions. Shown from a dictionary of common queries built into the extension — matching against what you type happens on-device. Nothing is transmitted anywhere; no external services are used.

Временно отключённые функцииTemporarily Disabled Features

Прогноз погоды и синхронизация между устройствами временно отключены в этой версии расширения — до появления собственной серверной инфраструктуры на территории РФ. Соответствующий интерфейс скрыт, а код, отправляющий запросы к этим сервисам, удалён из расширения полностью, а не временно деактивирован — эти функции сейчас не запрашивают согласие и физически не могут передать никакие данные. Планируем вернуть обе функции по мере роста числа пользователей, вместе с развёртыванием собственного сервера в РФ.

The weather forecast and cross-device sync are temporarily disabled in this version of the extension — pending our own server infrastructure located in Russia. The corresponding interface is hidden, and the code that sends requests to these services has been removed from the extension entirely, not just deactivated — these features currently do not request consent and cannot physically transmit any data. We plan to bring both features back as our user base grows, together with our own server in Russia.

Когда эти функции вернутся, для каждой снова появится отдельный переключатель в настройках, а согласие будет запрашиваться отдельно на каждую — этот документ будет обновлён заранее, до включения функций.

When these features return, each will again have its own toggle in settings, and consent will be requested separately for each — this document will be updated in advance, before the features are enabled.

Срок действия согласияDuration of Consent

Согласие на опциональные функции действует, пока функция включена в настройках. Согласие на основные функции действует, пока вы пользуетесь расширением. В обоих случаях — до момента удаления расширения, если наступит раньше.

Consent for optional features lasts as long as the feature is enabled in settings. Consent for core features lasts as long as you use the extension. In both cases, consent ends no later than when the extension is removed, if that happens sooner.

Как отозвать согласиеHow to Withdraw Consent

Опциональные функции — отключите переключатель в настройках в любой момент, это так же просто, как и включить его.Optional features — turn off the toggle in settings at any time; it's just as easy as turning it on.

Основные функции — отдельного тумблера нет; не используйте конкретную функцию (например, не нажимайте «Импортировать закладки») либо удалите расширение целиком.Core features — there is no separate toggle; simply don't use that particular feature (for example, don't click "Import bookmarks"), or remove the extension entirely.
