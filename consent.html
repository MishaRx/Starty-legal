Основные функции расширенияCore Features of the Extension

Эти функции работают как часть базовой работы расширения — согласие на них даётся самим фактом установки и использования расширения по его прямому назначению:

These features are part of the extension's basic operation — consent to them is given by the fact of installing and using the extension as intended:

Отображение значков сайтов. При сохранении ссылки её домен используется для получения иконки (favicon) сайта из локальной базы браузера (разрешение «favicon»). Обрабатывается локально, никуда не передаётся.Displaying site icons. When you save a link, its domain is used to fetch the site's icon (favicon) from the browser's local database (the "favicon" permission). Processed locally, never transmitted anywhere.

Импорт закладок. Если при первой настройке вы выбираете «Импортировать закладки» — расширение считывает заголовки и адреса ваших закладок, чтобы предложить выбрать, какие добавить как плитки. Обрабатывается локально, никуда не передаётся.Bookmark import. If you choose "Import bookmarks" during initial setup, the extension reads the titles and addresses of your bookmarks to let you pick which ones to add as tiles. Processed locally, never transmitted anywhere.

Подсказки недавних сайтов. При открытии формы добавления новой ссылки расширение может показать недавно закрытые вкладки как быстрые варианты. Обрабатывается локально, никуда не передаётся.Recent-site suggestions. When opening the add-link form, the extension may show recently closed tabs as quick options. Processed locally, never transmitted anywhere.

Опциональные функцииOptional Features

Каждая функция — строго добровольна, включается отдельным переключателем в настройках. Согласие даётся отдельно на каждую в момент, когда вы её включаете:

Each feature is strictly opt-in, enabled via its own toggle in settings. Consent is given separately for each one, at the moment you turn it on:

Прогноз погоды. Передача названия города, которое вы вводите текстом, в open-meteo.com и wttr.in — для отображения прогноза погоды.Weather forecast. Sending the city name you type to open-meteo.com and wttr.in — to display a weather forecast.

Синхронизация между устройствами. Передача сохранённых ссылок, папок, режимов, заметок и задач в инфраструктуру синхронизации вашего браузера — для доступа с разных устройств.Cross-device sync. Sending your saved links, folders, modes, notes, and tasks to your browser's sync infrastructure — for access across devices.

Подсказки при поиске. Текст, который вы вводите в строку поиска (по мере набора, до нажатия Enter), передаётся в suggestqueries.google.com для показа вариантов автодополнения. По умолчанию выключено.Search suggestions. Text you type into the search bar (as you type, before pressing Enter) is sent to suggestqueries.google.com to show autocomplete suggestions. Off by default.

Срок действия согласияDuration of Consent

Согласие на опциональные функции действует, пока функция включена в настройках. Согласие на основные функции действует, пока вы пользуетесь расширением. В обоих случаях — до момента удаления расширения, если наступит раньше.

Consent for optional features lasts as long as the feature is enabled in settings. Consent for core features lasts as long as you use the extension. In both cases, consent ends no later than when the extension is removed, if that happens sooner.

Как отозвать согласиеHow to Withdraw Consent

Опциональные функции — отключите переключатель в настройках в любой момент, это так же просто, как и включить его.Optional features — turn off the toggle in settings at any time; it's just as easy as turning it on.

Основные функции — отдельного тумблера нет; не используйте конкретную функцию (например, не нажимайте «Импортировать закладки») либо удалите расширение целиком.Core features — there is no separate toggle; simply don't use that particular feature (for example, don't click "Import bookmarks"), or remove the extension entirely.
