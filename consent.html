На что вы соглашаетесьWhat You're Agreeing To

В текущей версии (1.0) расширение обрабатывает данные исключительно локально, на вашем устройстве. Ни одна из функций ниже не отправляет данные куда-либо за пределы вашего браузера. Согласие на каждую даётся самим фактом использования соответствующей функции — отдельного действия (клика, галочки) не требуется, потому что ни одна из них не связана с передачей данных наружу.

In the current version (1.0), the extension processes data exclusively locally, on your device. None of the features below send data anywhere outside your browser. Consent to each is given by the fact of using that feature — no separate action (a click, a checkbox) is required, because none of them involve sending data outside.

Отображение значков сайтов. Домен сохранённой ссылки используется для запроса иконки из локальной базы фавиконок браузера (permission «favicon»). Домен не покидает устройство.Displaying site icons. The domain of a saved link is used to request an icon from the browser's local favicon database (the "favicon" permission). The domain never leaves the device.

Импорт закладок. Если вы сами нажимаете «Импортировать закладки» — расширение один раз считывает заголовки и адреса ваших закладок (permission «bookmarks», запрашивается браузером именно в этот момент, не при установке), чтобы предложить выбрать, какие добавить как плитки.Bookmark import. If you click "Import bookmarks" yourself, the extension reads the titles and addresses of your bookmarks once (the "bookmarks" permission, requested by the browser at that exact moment, not at install) to let you pick which ones to add as tiles.

Подсказки недавних сайтов. При открытии формы добавления новой ссылки расширение может показать недавно закрытые вкладки как быстрые варианты (permissions «sessions» и «tabs», запрашиваются в этот момент, не при установке).Recent-site suggestions. When opening the add-link form, the extension may show recently closed tabs as quick options (the "sessions" and "tabs" permissions, requested at that exact moment, not at install).

Подсказки при поиске. Текст, который вы вводите в строку поиска, сравнивается со словарём частых запросов, встроенным в само расширение. Ничего не передаётся, внешние сервисы не задействованы.Search-bar suggestions. Text you type into the search bar is matched against a dictionary of common queries built into the extension itself. Nothing is transmitted; no external service is involved.

Подбор ссылок при первой настройке. Сфера деятельности, которую вы указываете сами при первом запуске, используется один раз, локально, чтобы предложить релевантный набор стартовых ссылок.Initial link suggestions. The industry you specify yourself on first run is used once, locally, to suggest a relevant set of starter links.

Функции, которых пока нетFeatures That Don't Exist Yet

Ничего в этом разделе не работает в текущей версии. Мы не запрашиваем согласие на функции ниже прямо сейчас — он здесь только для прозрачности на будущее.
Nothing in this section is active in the current version. We are not requesting consent for the features below right now — this section exists only for forward-looking transparency.

Мы планируем добавить прогноз погоды и синхронизацию настроек между устройствами — после того как появится собственная серверная инфраструктура в РФ (разворачиваем по мере роста числа пользователей). Когда это произойдёт: обе функции будут добровольными, с отдельным переключателем для каждой, выключенным по умолчанию. Согласие на каждую будет запрашиваться отдельно, в момент включения — этот документ будет обновлён заранее, до того как функции появятся, с точным описанием того, что и куда передаётся.

We plan to add a weather forecast and cross-device settings sync — once we have our own server infrastructure in Russia (which we're setting up as Starty's user base grows). When that happens: both features will be opt-in, each with its own toggle, off by default. Consent for each will be requested separately, at the moment it's turned on — this document will be updated in advance, before the features launch, with an exact description of what is sent and where.

Срок действия согласияDuration of Consent

Согласие на функции из раздела 2 действует всё время, пока вы пользуетесь расширением, и прекращается, когда вы его удаляете. Если в будущем появятся функции из раздела 3, срок действия согласия на каждую из них будет отдельно описан при её запуске.

Consent to the features in section 2 lasts for as long as you use the extension and ends when you remove it. If the features in section 3 launch in the future, the duration of consent for each will be described separately at that time.

5

Как отозвать согласиеHow to Withdraw Consent

Поскольку ни одна текущая функция не передаёт данные за пределы вашего устройства, отзывать в строгом смысле нечего. Если вы не хотите, чтобы расширение выполняло конкретную функцию из раздела 2 — просто не используйте её (например, не нажимайте «Импортировать закладки»), либо удалите расширение целиком, чтобы прекратить любую обработку.

Because no current feature transmits data beyond your device, there is nothing to withdraw in the strict sense. If you don't want the extension to perform a specific feature from section 2, simply don't use it (for example, don't click "Import bookmarks"), or remove the extension entirely to stop all processing.
