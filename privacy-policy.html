2. Цели обработки персональных данных

Мы обрабатываем персональные данные исключительно для следующих целей:

  • Отображение прогноза погоды по городу, который пользователь указывает самостоятельно
  • Отображение значков (favicon) сайтов, ссылки на которые пользователь сохраняет в расширении
  • Синхронизация настроек и сохранённых ссылок между устройствами пользователя (только если пользователь сам включил эту функцию)
  • Улучшение продукта на основе агрегированной статистики использования (только если пользователь сам включил соответствующую функцию в настройках)

Мы не используем персональные данные для целей, не указанных выше — в частности, не передаём и не продаём их третьим лицам для маркетинга, не используем для кредитного скоринга и не строим индивидуальные профили пользователей.

4. Категории субъектов персональных данных

Пользователи расширения Starty — физические лица, установившие расширение в свой браузер.

5. Правовое основание обработки

Обработка осуществляется на основании согласия пользователя (ст. 9 152-ФЗ) — оформляется отдельным документом при первом запуске расширения (см. Согласие на обработку персональных данных). Пользователь может в любой момент отказаться от обработки, отключив соответствующие функции в настройках расширения — это не влияет на работу остальных функций продукта.

6. Способы и сроки обработки и хранения

  • Название города и настройки хранятся локально в браузере пользователя (chrome.storage), пока пользователь не удалит расширение или не очистит данные вручную
  • При включённой синхронизации — данные дополнительно хранятся в инфраструктуре синхронизации браузера пользователя, в соответствии с политиками конфиденциальности этих сервисов
  • Обезличенные события статистики хранятся в Яндекс.Метрике в соответствии с политикой хранения данных Яндекса

Мы не храним резервных копий персональных данных пользователей на собственных серверах — на данный момент (версия 1.0) собственных серверов, обрабатывающих персональные данные, у Starty нет.

7. Порядок уничтожения персональных данных

  • Данные, хранящиеся локально, уничтожаются при удалении расширения пользователем или очистке данных браузера
  • Отключение соответствующей функции в настройках останавливает дальнейшую передачу данных
  • Для запроса на уничтожение данных, переданных через синхронизацию или Яндекс.Метрику — обратиться по контактному email, указанному в разделе 1

8. Права субъекта персональных данных

Пользователь имеет право:

  • Получить информацию об обработке своих персональных данных
  • Потребовать уточнения, блокирования или уничтожения неточных или незаконно обрабатываемых данных
  • Отозвать согласие на обработку персональных данных в любой момент
  • Обратиться с жалобой в Роскомнадзор, если считает, что его права нарушены

Для реализации этих прав — обратиться по контактному email, указанному в разделе 1.

9. Как отозвать согласие

Отозвать согласие можно так же просто, как и дать его: отключить соответствующий тумблер в настройках расширения («Синхронизация», «Статистика использования»), либо удалить расширение из браузера — это остановит любую дальнейшую обработку данных Starty.

10. Меры по обеспечению безопасности персональных данных

  • Данные о городе для погоды передаются по защищённому протоколу (HTTPS) непосредственно во внешние сервисы — без промежуточного хранения на серверах Starty
  • Расширение не запрашивает и не хранит пароли, платёжные данные или иную чувствительную информацию
  • Код расширения проходит проверку Chrome Web Store перед публикацией

11. Условия прекращения обработки

Обработка прекращается при достижении цели обработки, отзыве согласия пользователем, обнаружении неправомерной обработки, либо при удалении расширения.

12. Изменения в политике

Мы можем обновлять эту политику по мере развития продукта. Актуальная версия всегда доступна по этой ссылке. Дата последнего обновления указана в начале документа.
