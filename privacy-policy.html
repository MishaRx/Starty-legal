Цели обработки персональных данныхPurposes of Processing

Мы обрабатываем персональные данные исключительно для следующих целей:

We process personal data exclusively for the following purposes:

  • Отображение прогноза погоды по городу, который пользователь указывает самостоятельно
  • Отображение значков (favicon) сайтов, ссылки на которые пользователь сохраняет в расширении — обрабатывается локально, не передаётся третьим лицам
  • Показ подсказок при вводе запроса в строку поиска (только если пользователь сам включил эту функцию — по умолчанию выключена)
  • Предложение быстрого добавления ссылок на основе закладок браузера (только при первой настройке, по выбору пользователя) и недавно закрытых вкладок (при открытии формы добавления ссылки)
  • Подбор релевантных ссылок на основе указанной пользователем сферы деятельности при первой настройке расширения
  • Синхронизация настроек и сохранённых ссылок между устройствами пользователя (только если пользователь сам включил эту функцию)
  • Displaying a weather forecast for a city the user enters themselves
  • Displaying site icons (favicons) for links the user saves in the extension — processed locally, not shared with third parties
  • Showing suggestions while typing in the search bar (only if the user has enabled this feature themselves — off by default)
  • Offering quick-add suggestions based on browser bookmarks (only during initial setup, by user choice) and recently closed tabs (when opening the add-link form)
  • Suggesting relevant links based on the industry the user specifies during initial setup
  • Syncing settings and saved links across the user's devices (only if the user has enabled this feature themselves)

Мы не используем персональные данные для целей, не указанных выше — в частности, не передаём и не продаём их третьим лицам для маркетинга, не используем для кредитного скоринга и не строим индивидуальные профили пользователей.

We do not use personal data for any purpose other than those listed above — in particular, we do not share or sell it to third parties for marketing, use it for credit scoring, or build individual user profiles.

Правовое основание обработкиLegal Basis for Processing

Обработка осуществляется на основании согласия пользователя (ст. 9 152-ФЗ) — оформляется отдельным документом «Согласие на обработку персональных данных», в котором отдельно описаны основные функции (согласие даётся фактом использования расширения по назначению) и опциональные функции, включаемые отдельными тумблерами (погода, синхронизация, подсказки при поиске). Пользователь может в любой момент прекратить обработку данных для опциональных функций, отключив соответствующий тумблер в настройках.

Processing is carried out on the basis of user consent (Art. 9 of Federal Law No. 152-FZ "On Personal Data") — documented separately in the "Consent to Personal Data Processing", which distinguishes core features (consent given by the fact of using the extension as intended) from optional features enabled via individual toggles (weather, sync, search suggestions). The user may stop processing for any optional feature at any time by turning off the corresponding toggle in settings.

Способы и сроки обработки и храненияMethods and Retention Periods

  • Название города и настройки хранятся локально в браузере пользователя (chrome.storage), пока пользователь не удалит расширение или не очистит данные вручную
  • При включённой синхронизации — данные дополнительно хранятся в инфраструктуре синхронизации браузера пользователя
  • The city name and settings are stored locally in the user's browser (chrome.storage) until the user removes the extension or manually clears the data
  • If sync is enabled, the data is additionally stored in the user's browser sync infrastructure

Мы не храним резервных копий персональных данных пользователей на собственных серверах — на данный момент (версия 1.0) собственных серверов, обрабатывающих персональные данные, у Starty нет.

We do not keep backup copies of users' personal data on our own servers — as of now (version 1.0), Starty has no servers of its own that process personal data.

Порядок уничтожения персональных данныхData Deletion Procedure

  • Кнопка «Удалить все данные Starty» в настройках (Синхронизация и бэкап) — безвозвратно удаляет все локальные данные (ссылки, режимы, заметки, задачи, кастомные изображения, кэш значков сайтов, расписание) и, если была включена синхронизация, соответствующие данные в инфраструктуре синхронизации вашего браузера — без необходимости удалять расширение целиком
  • Данные, хранящиеся локально, также уничтожаются при удалении расширения пользователем или очистке данных браузера
  • Отключение соответствующей функции в настройках останавливает дальнейшую передачу данных
  • Данные синхронизации, помимо кнопки выше, можно удалить полностью, отключив синхронизацию и очистив данные синхронизации через настройки вашего браузерного аккаунта — это может понадобиться, если синхронизация была включена на устройстве, к которому у вас больше нет доступа
  • The "Delete all Starty data" button in settings (Sync & Backup) permanently deletes all local data (links, modes, notes, tasks, custom images, favicon cache, schedule) and, if sync was enabled, the corresponding data in your browser's sync infrastructure — without needing to remove the extension entirely
  • Locally stored data is also destroyed when the user removes the extension or clears browser data
  • Turning off a feature in settings stops any further transmission of data
  • Beyond the button above, sync data can be fully removed by disabling sync and clearing sync data through your browser account settings — this may be needed if sync was enabled on a device you no longer have access to
